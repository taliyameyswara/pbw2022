HyperText Markup Language

HTML atau HyperText Markup Language merupakan sebuah bahasa markah untuk membuat halaman web

Attributes HTML

Atribut adalah kata khusus yang berada di dalam tag pembuka. Atribut juga disebut sebagai modifier yang akan menentukan perliaku dari elemen. Atribut dapat ditambahkan pada elemen manapun. Ada juga elemen yang mewajibkan menggunakan atribut seperti elemen < a >, < img >, < video >, dll

Heading HTML

Heading merupakan element atau tag HTML yang berfungsi untuk menunjukkan bagian penting pada halaman web. Element tag heading ini memiliki enam tingkatan yang berurutan yaitu h1-h6 dari besar ke kecil. Berikut adalah contoh penggunaan heading:
